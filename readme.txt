Online Data Collection Management ---- YouTube API



For our research whe used the official YouTube API to scrape data.
 The API scrapes the following data from YouTube:
- Video snippets
- Video Statatistics
- Channel Statistics

This data will allow other researchers to perform multiple types of research.
Before the API can be run, first the researcher should request an API key from YouTube. 
More information about this can be found in the PDF file: ODCM links and then click on the 'getting started link'.
